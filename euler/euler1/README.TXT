Sample C and Python implementations for Euler Project problem 1.

If python is installed, run with:
$python euler1.py
Or just
$./euler1.py

To compile and the C version:
$gcc -o euler1 euler1.c
$./euler1

If gcc or python are not installed, use your distributions package
manager to install them. (Software Center in Ubuntu, apt-get install, 
yum install, etc.)
